# EVALUACION 1 FRONT END

# Mi primer Portafolio Web 

¡Bienvenido a mi proyecto para la prueba numero 1 de front end de desarrollo web! Este es un portafolio personal diseñado para mostrar mi perfil profesional, mis habilidades técnicas y los proyectos realizados durante mi formación en **INACAP**.

## Sobre el Proyecto
Este sitio fue construido desde cero utilizando buenas prácticas de desarrollo web, con un enfoque en diseño limpio y responsive.

### Tecnologías utilizadas:
* **HTML5**: Estructura semántica completa.
* **CSS3**: Diseño personalizado, uso de Flexbox y CSS Grid.
* **Git/GitHub**: Actualizaciones mediante commits.

##  Secciones del sitio
1. **Navbar y estructura inicial**: Presentación principal con impacto visual.
2. **Sobre Mí**: Biografía profesional y datos personales.
3. **Habilidades y tabla**: Listado técnico de conocimientos.
4. **Galeria de proyectos**: Galería de trabajos realizados.
5. **Formulario de contacto**: Formulario funcional para comunicación.
6. **Footer**: Pie de pagina

##  Instalación y Vista Previa
Si quieres ver el proyecto de forma local, sigue estos pasos:
1. Clona el repositorio:
   `git clone https://github.com/juako18102002-prog/prueba1`
2. Abre el archivo `index.html` en tu navegador preferido.
